# Product Guidelines: Social Media Trend Analyzer

## Prose Style
- **Clarity and Precision:** Use simple, direct language to explain technical data insights.
- **Tone:** Professional, analytical, and supportive of data-driven decision-making.
- **Terminology:** Consistently use terms like "Themes," "Correlations," and "Trending Keywords" as defined in the PRD.

## Design Principles
- **Data-First:** Prioritize visual clarity for charts and tables. Ensure data is the central focus of the dashboard.
- **Consistency:** Use Tailwind CSS to maintain consistent spacing, typography, and color schemes across all screens.
- **Modern Aesthetic:** A clean, minimalist "Mission Control" feel, aligning with the case study's high-stakes narrative.

## User Experience (UX)
- **Guided Workflow:** Ensure the path from data upload to report generation is intuitive and requires minimal user input.
- **Instant Feedback:** Provide visual indicators during data processing and report generation.
- **Responsive Design:** Support both mobile and desktop viewports, using the provided Stitch UI templates as a baseline.

## Technical Standards
- **Performance:** Optimize data cleaning and clustering to provide near-instant results for moderately sized datasets.
- **Security:** Maintain data privacy by ensuring all processing is handled securely and temporary data is cleared after analysis.
